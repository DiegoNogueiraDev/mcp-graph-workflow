# Support

## How to get help

- **Documentation** — Start with the [README](README.md) and [CLAUDE.md](CLAUDE.md) for architecture and conventions.
- **GitHub Discussions** — Ask questions, share ideas, and connect with the community at [Discussions](https://github.com/DiegoNogueiraDev/mcp-graph-workflow/discussions).
- **GitHub Issues** — Found a bug or want to request a feature? Open an [issue](https://github.com/DiegoNogueiraDev/mcp-graph-workflow/issues) using the appropriate template.

## What NOT to use issues for

- General "how do I..." questions — use Discussions instead.
- Security vulnerabilities — see [SECURITY.md](SECURITY.md) for responsible disclosure.

## Response times

This is an open-source project maintained by volunteers. We do our best to respond promptly, but please allow a few days for a reply.
